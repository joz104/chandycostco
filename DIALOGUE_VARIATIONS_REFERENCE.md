# Dialogue Variations Quick Reference

A side-by-side comparison of all dialogue variations for easy reference.

---

## INTRO VARIATIONS

### Variation A (Original - Plays 0-2)
Classic earnest Chandler saying goodbye to Mama.

**Tone**: Heartfelt, sincere
**Key Lines**:
- "I dream of the Freedom Glizzy. $1.50 with a soda."
- "I am fast, Mama. Like the wind."

### Variation B (Plays 3-5)
Chandler is a repeat offender, Mama is exasperated.

**Tone**: Comedic, familiar
**Key Lines**:
- "Again? This is the third time this week!"
- "Fine. But this time, bring me back the BIG churro."

### Variation C (Plays 6-8)
Chandler treats it like a sacred prophecy.

**Tone**: Epic, mystical
**Key Lines**:
- "The prophecy spoke of the Sacred Glizzy."
- "May the Costco gods watch over you."

## ENDING VARIATIONS (Coin-Based)

### BROKE (0 coins = $0.00)
**Tone**: Complete failure, maximum despair
**Summary**: Chandler has NO MONEY at all. He didn't collect a single coin.
**Result**: Kicked out immediately.
**Key Line**: "You came all this way with NO MONEY?"

---

### DESPERATE (1-2 coins = $0.25-$0.50)
**Tone**: Pathetic begging
**Summary**: Has pocket change but nowhere near enough.
**Result**: Called a beggar, removed by security.
**Key Line**: "Should've crossed with more quarters."

---

### CLOSE (3-5 coins = $0.75-$1.25)
**Tone**: Heartbreaking, so close yet so far
**Summary**: Has $1.25 - just 25 cents short!
**Result**: No mercy, kicked out despite being close.
**Key Line**: "So close. Yet so far."

---

### EXACT (6 coins = $1.50)
**Tone**: Perfect victory, earned success
**Summary**: Exactly the right amount! Perfect change!
**Result**: SUCCESS! Gets the Freedom Glizzy!
**Key Line**: "THE FREEDOM GLIZZY! IT IS BEAUTIFUL!"

---

### TIP (7-9 coins = $1.75-$2.25)
**Tone**: Generous customer, rewarded kindness
**Summary**: Has extra money and tips the cashier!
**Result**: Gets extra napkins as reward.
**Key Line**: "Keep it! You work hard!"

---

### RICH (10-14 coins = $2.50-$3.50)
**Tone**: Big spender energy
**Summary**: Orders TWO combos with a dollar tip!
**Result**: Gets extra churros on the house!
**Key Line**: "I am a MAN OF WEALTH now!"

---

### LEGEND (15+ coins = $3.75+)
**Tone**: Epic hero status
**Summary**: Orders THREE combos AND a pizza! $10 payment!
**Result**: Gets FREE MEMBERSHIP CARD! Manager called over!
**Key Line**: "MAMA! I'M NEVER LEAVING COSTCO!"

## Quick Selection Guide

### For Testing Specific Variations:

```javascript
// Test NPC variations
getSpecificVariation('L2', 0); // Coyote A (standard)
getSpecificVariation('L2', 1); // Coyote B (menacing)
getSpecificVariation('L2', 2); // Coyote C (mystical)

// Test endings with coin counts
getEndingVariation(0);  // BROKE
getEndingVariation(2);  // DESPERATE
getEndingVariation(5);  // CLOSE
getEndingVariation(6);  // EXACT (perfect!)
getEndingVariation(8);  // TIP
getEndingVariation(12); // RICH
getEndingVariation(20); // LEGEND
```

### Natural Progression:

**First Playthrough** (0-2 games played):
- Variation A dialogue everywhere
- Ending depends on coins collected

**Second Cycle** (3-5 games played):
- Variation B dialogue (funnier, alternate personalities)
- New jokes and references

**Third Cycle** (6-8 games played):
- Variation C dialogue (most creative)
- Easter eggs and meta-humor

**After That**:
- Cycles back through A → B → C → A → B → C...

---

## Recommended Playtesting Path

1. **First run**: Collect 0 coins → See BROKE ending
2. **Second run**: Collect 5 coins → See CLOSE ending (heartbreaking!)
3. **Third run**: Collect exactly 6 → See EXACT ending (satisfaction!)
4. **Fourth run**: Notice dialogue has changed to Variation B
5. **Fifth run**: Collect 15+ coins → See LEGEND ending (epic!)
6. **Sixth run**: Notice dialogue changed to Variation C

This gives players a progression:
- Failure → Near success → Success → Mastery
